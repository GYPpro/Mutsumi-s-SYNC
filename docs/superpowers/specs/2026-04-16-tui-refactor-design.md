# TUI 重构设计方案

## 概述

将现有的 textual TUI 替换为基于 prompt_toolkit 的命令行交互界面，提供类似服务器后端的操作体验。

## 界面布局

```
┌─────────────────────────────────────────────────────┐
│  [AI: ●] [NapCat: ●] [MSG: 42] [Uptime: 00:05:32]  │
├─────────────────────────────────────────────────────┤
│                                                     │
│           (日志/命令输出区域)                       │
│                                                     │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  ❯ config get model.provider                       │
└─────────────────────────────────────────────────────┘
```

## 配色方案 (Vim Solarized Dark 风格)

| 元素 | 颜色 | Hex |
|------|------|-----|
| 背景 | 深灰 | `#002b36` |
| 前景 | 浅灰 | `#839496` |
| 强调色 | 青色 | `#2aa198` |
| 警告 | 黄色 | `#b58900` |
| 错误 | 红色 | `#dc322f` |
| 成功 | 绿色 | `#859900` |
| 状态栏背景 | 深青 | `#073642` |
| 边框 | 灰 | `#586e75` |

## 状态栏

```
[AI: ●] [NapCat: ●] [MSG: 42] [Uptime: 00:05:32]
```

- `●` 绿色 (#859900) = 在线
- `○` 红色 (#dc322f) = 离线

## 命令行

- 提示符: `❯` (青色 #2aa198)
- 光标: 方块，光标颜色为强调色
- 选中补全项: 背景 `#073642`

## 命令设计

### 配置管理
| 命令 | 功能 |
|------|------|
| `config list` | 列出所有配置 |
| `config get <key>` | 获取配置值 |
| `config set <key> <value>` | 修改配置 |
| `config reload` | 重载配置 |

### 日志操作
| 命令 | 功能 |
|------|------|
| `logs tail [-n N]` | 查看最近 N 行 |
| `logs grep <pattern>` | 搜索日志 |
| `logs level <LEVEL>` | 设置日志级别 |

### 系统
| 命令 | 功能 |
|------|------|
| `status` | 显示系统状态 |
| `stats` | 显示统计 |
| `reload` | 重连服务 |
| `conversations` | 进入对话浏览 |
| `help` | 显示帮助 |
| `exit` | 退出 |

## 语法补全

- **命令名补全**: Tab 补全 `logs`, `config`, `status` 等
- **子命令补全**: `logs <Tab>` → `tail`, `grep`, `level`
- **参数补全**: `-n<Tab>` → `-n 20`
- **历史**: 上/下方向键遍历最近 100 条

## 对话浏览模式

`conversations` 进入:
- 用户列表: `← →` 切换用户
- 轮次导航: `↑ ↓` 切换轮次
- 详情查看: `Enter` 显示 Tool Calls
- 返回: `Esc`
